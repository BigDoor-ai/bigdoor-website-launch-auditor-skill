# Usage

## Initialize in a website project

```bash
bd-audit init
```

## Export agent instructions

```bash
bd-audit export-agent cursor
bd-audit export-agent codex
bd-audit export-agent claude-code
```

## Ask your coding agent

```txt
Use the Bigdoor Website Launch Auditor Skill.
Do not modify code.
Generate a complete launch-readiness report.
```

## Run checks

```bash
bd-audit check-project
bd-audit check-url https://example.com
bd-audit audit
```
